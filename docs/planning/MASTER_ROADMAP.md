# AI Hospital Assistant Alliance - Master Roadmap

## Product Layers
- Platform: Backend, API, Auth, Database, Monitoring, Integrations
- Web: Admin, Doctor, Nurse, Pharmacy, Reports, PACS, Labs
- App: Doctor mobile, Nurse mobile, Patient mobile
- AI: Clinical Brain, Smart Pharmacy, Risk Scoring, Report Intelligence

## Current Product Level
- Level 3: AI Medical Platform

## Core Rule
Any new feature must be classified into:
1. Shared Logic
2. Platform/API
3. Web UI
4. App UI
5. AI Logic
6. Data/Storage
7. Security/Permissions
8. Monitoring/Analytics

## Delivery Rule
No feature is considered complete unless reviewed for:
- Web impact
- App impact
- Platform impact
- API contract
- Role permissions
- Performance impact
